## Worksheet 6: The ADHD Freeze Protocol Card
*Laminated-ready — print, cut, and keep this where you work*

```
┌─────────────────────────────────────────────┐
│         ADHD FREEZE PROTOCOL                │
│         Wellness Within Reach               │
├─────────────────────────────────────────────┤
│                                             │
│  Step 1: NAME IT (30 sec)                   │
│  "I am in freeze. This is neurological.     │
│   It will pass."                            │
│                                             │
│  Step 2: MOVE (2 min)                       │
│  Walk / jumping jacks / shake it out        │
│  Dance to one song. Any movement counts.    │
│                                             │
│  Step 3: REDUCE (60 sec)                    │
│  Close tabs. Silence phone. Headphones on.  │
│                                             │
│  Step 4: ONE TINY THING (now)               │
│  Not the project. Not the goal.             │
│  Open one file. Write one sentence.         │
│  Send one email. Just one.                  │
│                                             │
│  Step 5: GIVE YOURSELF CREDIT               │
│  You started from freeze. That is real.     │
│                                             │
└─────────────────────────────────────────────┘
```
